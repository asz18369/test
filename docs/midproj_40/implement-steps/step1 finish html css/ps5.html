Seagate 2TB 外接式遊戲硬碟是根據 PlayStation® 官方授權計畫的規範所開發，並可在獲得 Sony Interactive Entertainment 最終產品核准後與 PlayStation®5 搭配使用。

外接式 USB 硬碟相容性

PS5 主機可將相容的 PS4 和 PS5 遊戲儲存至支援 SuperSpeed USB* 的 USB 硬碟。這表示您可節省內部超高速 SSD 固態硬碟的空間，並透過從外接式 USB 硬碟重新安裝 PS5 遊戲來節省時間，而無須重新下載遊戲或另外從光碟安裝。

當您準備遊玩儲存在外接式 USB 硬碟的 PS5 遊戲時，只要透過內容保存庫的選項選單將遊戲傳回內部的 SSD 固態硬碟即可。您可以直接從外接式 USB 硬碟或內部的 SSD 固態硬碟遊玩 PS4 遊戲。

*250 GB以上、8 TB以下的容量。
